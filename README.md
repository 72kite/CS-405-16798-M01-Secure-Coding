# CS-405 Secure Coding
    
    ## Overview
    
    This repository contains artifacts related to defensive coding practices, security policy documentation, and vulnerability assessments, with a focus on their integration into the Software Development Life Cycle (SDLC).
    
    ## Repository Structure
    
    -   `CS_405_Security_Policy.docx`: The main security policy document.
    -   `CS 405 Security Policy Presentation Script.docx`: The script for the security policy presentation.
    -   `Green-Pace-Security-Policy-Guide.pptx`: The presentation slides for the security policy guide.
    -   `Programs Code).zip`: Source code files related to the project.
    -   `Project 2_Presentation_.mp4`: The video recording of the project presentation.
    
    ## Key Topics
    
    -   Software Development Life Cycle (SDLC)
    -   Defensive Coding
    -   Security Policy Development
